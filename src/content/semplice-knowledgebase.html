Challenge

The knowledgebase was not responsive and lacked an intuitive user interface. The Semplice Labs team believe this led to more support tickets being opened that could have otherwise been solved through the user finding the information within the knowledgebase.

Solution

Make the knowledgebase responsive, so that it is accessible to all users. Redesign the interface to allow users to find information faster and know their location within the knowledgebase. Create a dashboard as the landing page, which facilitates users in finding vital information for using Semplice.
